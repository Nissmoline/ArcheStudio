# ARCHÉ STUDIO

Luxury carpentry and bespoke furniture website built with React and Vite.

## Scripts

```bash
npm install
npm run dev
npm run build
```

Local development runs on:

```bash
http://127.0.0.1:3000
```

## Deploy

The project is configured for Vercel with `vercel.json`.

Build command:

```bash
npm run build
```

Output directory:

```bash
dist
```
